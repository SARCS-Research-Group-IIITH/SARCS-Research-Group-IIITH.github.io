# SARCS Research Group Website

Official website for the Sustainable, Advanced and Robust Computing Systems (SARCS) Research Group at IIIT Hyderabad.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content

All website content is managed through JSON files in the `/data` directory.
